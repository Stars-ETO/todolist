### 技术栈选型

#### 前端技术栈



*   **框架核心**：Vue 3（采用 Composition API，更适合复杂状态管理）

*   **路由管理**：Vue Router（实现页面跳转和权限控制）

*   **状态管理**：Pinia（替代 Vuex，更简洁的 API，支持 TypeScript）

*   **UI 组件库**：Element Plus（与 Vue 3 适配，提供丰富的表单、表格组件）

*   **HTTP 客户端**：Axios（处理与后端的 API 交互）

*   **图表工具**：ECharts（用于数据统计模块的可视化展示）

*   **样式解决方案**：Tailwind CSS（快速构建响应式界面）

#### 后端技术栈



*   **Web 框架**：FastAPI（高性能异步框架，自动生成 API 文档）

*   **数据库**：PostgreSQL（关系型数据库，适合存储用户、任务等结构化数据）

*   **ORM 工具**：SQLAlchemy（处理数据库交互，支持异步操作）

*   **认证方案**：JWT（JSON Web Token，用于用户身份验证）

*   **文件存储**：本地文件系统（存储用户上传的任务附件）
